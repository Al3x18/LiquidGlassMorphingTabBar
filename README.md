# LiquidGlassMorphingTabBar

A SwiftUI example project showcasing a morphing tab bar animation inspired by Apple's Liquid Glass design language.

## Overview

This project demonstrates how to build a custom morphing tab bar in SwiftUI with smooth transitions, animated shape transformations, and expandable content areas.

The implementation is based on the excellent tutorial by Kavsoft at this [YouTube](https://www.youtube.com/watch?v=Nj285uRnrPQ&list=LL&index=2) link
